\# Education Technology Impact Analysis



A data engineering and analytics project analyzing the relationship between educational technology access and student outcomes across U.S. public schools.



\## 🏗️ Project Architecture



\## 📊 Planned Outputs



\- State-level performance trends (NAEP)

\- Technology access vs outcomes analysis

\- District-level comparisons

\- Interactive Tableau dashboard



\## Project Objective



This project investigates whether districts or states with higher levels of technology access show different trends in student achievement, absenteeism, and learning outcomes over time.



\## Current Scope



\- Time range: 2014–2024, or latest available data

\- Geography: State and district-level analysis

\- Focus areas:

&#x20; - Technology access

&#x20; - Academic performance

&#x20; - Student engagement

&#x20; - Learning outcomes



\## Data Sources



\- NCES Common Core of Data (CCD)

\- NAEP Nation’s Report Card

\- Civil Rights Data Collection (CRDC)
